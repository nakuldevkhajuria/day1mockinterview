        Solution 4:
            An execution context consists of two things:

            1. A reference to the variable object (VO) that represents the current scope of the code.
            2. A reference   to the value of the this keyword, which is determined by how the code is 
            called.


            types of Execution context:
            1. global Execution context.
            2. function execution context
            3. eval execution context

            In addition to these three types of execution contexts, there is also a concept called the 
            "execution context stack," which is a stack of execution contexts that represents the order in which functions are called.


        Solution 5:
            In summary, a first-class function in JavaScript is a function that can be stored in a variable, 
            passed as an argument to a function, returned as a value from a function, and treated like any other object in the language.

        Solution 6:
        Closures are often used in JavaScript to create private variables and methods. For example, a 
        closure can be used to define an object that has a private property, and a public method that can 
        access and modify that property.
